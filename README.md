# DesafioTech

Esse projeto tem como finalidade o desenvolvimento de uma SPA que consome a API da Marvel e retorna uma quantidade limitada de personagens.

Para executar o projeto será necessário o download dos arquivos disponíveis neste repositório,  após realizar o download execute o arquivo Index.html no seu navegador.

O projeto foi desenvolvido utilizando HTML 5, CSS3 e JavaScript. O JS fica responsável por buscar os personagens na API e exibir em cards, o CSS por sua vez organiza e estiliza as informações, os cards possuem a descrição do personagem no seu verso, que pode ser visto ao passar do mouse.
